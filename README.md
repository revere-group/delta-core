**Delta Core** is a robust, lightweight network management core built from the ground up for **Spigot 1.21**. It serves as a foundational infrastructure for Minecraft networks, supplying the essential tools required to handle server-wide features, player administration, and structural coordination efficiently.

> **Note:** Delta Core was originally designed and used for one of Revere's legacy Minecraft servers. This project serves as a base framework; technical support has officially ended, and it is provided purely under our standard use terms and agreements.

## Features

* **Chat & Conversation Systems** – Features fully toggleable chat formats, specialized staff message handling while in vanish, and direct private messaging configurations.
* **Rank & Grant Systems** – Includes a built-in rank hierarchy with automated permission refreshing across the entire network upon rank updates.
* **Punishment System** – Essential moderation tools including toggleable join/leave notifications and automated administration logging.
* **Server & Network Management** – Streamlined modules for syncing server actions, executing cross-server adjustments, and managing multi-server dependencies.
* **Visuals & Display** – Built-in, optimized custom Scoreboard, TabList service, and PlaceholderAPI hooks.
* **Network Essentials** – Pre-configured sub-features including a global Cooldown tracker, automatic Tips system, basic Economy/Coins management, a TPA system, dynamic Tags, Shop interfaces, and integrated Social features.


## Installation

1. Clone this repository to your local machine.
2. Open the project in your preferred IDE and build it to generate `Delta.jar`.
3. Place the compiled `Delta.jar` into your server’s `plugins` directory.
4. Start your server. After startup, configure your database settings in `database.yml`.
5. Restart the server and once fully loaded — you're ready to go!


## Usage

Delta Core functions as a unified network base. After installation, locate the configuration folder within your server's `plugins` directory to fine-tune features like chat structures, rank permissions, database connectivity, and scoreboard layouts to align with your network's requirements.
